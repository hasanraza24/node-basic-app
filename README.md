# Node Basic App

Node Basic App is a simple node which can be used as a boilerplate for your node app. 

## Library Used

Express -> Framework.
Mongoose -> ODM to connect to MongoDB.
Joi -> To validate inputs.
JWT -> To create token for authorization purpose.
